# Blog frontend project

Implementing a blog front-end using Next.js.

## Development Environment Setup

### Copy .env

```shell
cp .env.example .env
```

### Install dependencies

```shell
npm install
```

### Start development server

```shell
npm run dev
```

### Start Storybook

```shell
npm run storybook
```
